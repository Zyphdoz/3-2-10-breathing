### 3-2-10 Breathing visualization
This is a simple visualization tool to help learn the 3-2-10 breathing technique.
3-2-10 is a breathing technique that can enhance cognitive performance by stimulating the parasympathetic nervous system. If you would like more information on how that works, I recommend watching this 10 minute video http://youtube.com/watch?v=wvVsOGSO2Ik

This tool is hosted at https://zyphdoz.github.io/3-2-10-breathing/
